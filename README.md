# Kayo Sports Picture-in-Picture Force
<img src="icon.png" alt="ClipFlowLogo" width="128">

A lightweight Google Chrome extension that forces native Picture-in-Picture (PiP) mode on Kayo Sports, bypassing player restrictions.

## Features
- Overrides Kayo's built-in block on browser Picture-in-Picture.
- Runs entirely locally via Manifest V3 with minimal permissions (`activeTab` and `scripting`).
- 100% free and open-source.

## Installation Instructions

Since this extension is not hosted on the Chrome Web Store, you can install it manually using Chrome's Developer Mode:

1. **Download the code:** Click the green **Code** button at the top right of this GitHub page and select **Download ZIP**. Extract the ZIP file to a folder on your computer.
2. **Open Extensions page:** Open Google Chrome and navigate to `chrome://extensions/` in your address bar.
3. **Enable Developer Mode:** In the top-right corner of the Extensions page, toggle the **Developer mode** switch to **ON**.
4. **Load the Extension:** Click the **Load unpacked** button in the top-left corner.
5. **Select Folder:** Select the unzipped folder containing the `manifest.json` and `background.js` files.

## How to Use
1. Click the **Extensions puzzle piece icon** in your Chrome toolbar and **Pin** the "Kayo PiP Force" extension.
2. Go to [Kayo Sports](https://kayosports.com.au) and start playing any live stream or replay.
3. Click the extension icon in your toolbar. The video will instantly pop out into a floating, resizable window!
